=head1 NAME

OSSL_CMP_MSG_http_perform,
- implementation of HTTP transfer for CMP messages

=head1 SYNOPSIS

 #include <openssl/cmp.h>

 OSSL_CMP_MSG *OSSL_CMP_MSG_http_perform(OSSL_CMP_CTX *ctx,
                                         const OSSL_CMP_MSG *req);

=head1 DESCRIPTION

This is the API for tranferring CMP (Certificate Management Protocol) messages
over HTTP(S) with OpenSSL, implementing RFC 6712.

OSSL_CMP_MSG_http_perform() sends the given PKIMessage req to the CMP server
specified in B<ctx>.
It uses the transfer callback function set via L<OSSL_CMP_CTX_set_http_cb(3)>,
its optional argument that may be set via L<OSSL_CMP_CTX_set_http_cb_arg(3)>,
as well as the server, port, path, proxy, proxy port, and message timeout
parameters as far as present in the CMP context B<ctx>.

=head1 NOTES

CMP is defined in RFC 4210 (and CRMF in RFC 4211).
HTTP Transfer for CMP is defined in RFC 6712.

=head1 RETURN VALUES

OSSL_CMP_MSG_http_perform() returns the received CMP response or NULL no error.
